# 实验室试剂配方计算器

一个专为植物病理学实验室设计的试剂配方计算工具，帮助研究人员快速计算各种培养基、缓冲液和反应体系的配制用量。

## 功能特点

- **培养基配方**：PDA/PDB、MM最小培养基、TSB/TSA、LB/LBA、R2A、KB、1/2 MS、V8等
- **消毒试剂**：HgCl₂、NaClO等消毒液配制与稀释计算
- **菌悬液制备**：Bm分生孢子悬浮液、细菌接种悬浮液浓度调节
- **代谢物处理**：12HEA等代谢物的母液配制与工作液稀释
- **母液配制**：Tween-20、常用抗生素（10种）、PBS缓冲液
- **PCR反应**：Taq/Phusion反应体系、dNTP Mix配制、琼脂糖凝胶
- **备忘录**：常用器皿与培养基/基质添加量速查表
- **全局搜索**：支持跨标签页搜索所有试剂、培养基和配方
- **液体/固体切换**：培养基支持液体和固体版本一键切换

## 包含的培养基

| 培养基 | 用途 |
|--------|------|
| PDA/PDB | *Bipolaris maydis* 培养与产孢 |
| MM最小培养基 | *Sphingobium* 碳源利用实验 |
| TSB/TSA | *Sphingobium* 常规培养 |
| LB/LBA | 通用细菌培养基 |
| R2A | 环境微生物专用低营养培养基 |
| KB | 假单胞菌及植物相关细菌培养 |
| 1/2 MS | 植物组织培养与种子萌发 |
| V8 | 卵菌和真菌培养基 |

## 使用方法

直接在浏览器中打开 `index.html` 文件即可使用，无需安装任何依赖。
